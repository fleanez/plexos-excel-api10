# plexos-excel-api10


## Usage
Plexos-Excel-api.exe [CONFIG_EXCEL_FILE] [CONFIG_EXCEL_SHEET] [PLEXOS_XML_FILE] (optional)[SUFFIXE]
* [CONFIG_EXCEL_FILE]     Path to Excel config file");
* [CONFIG_EXCEL_SHEET]    Excel's sheet name (case sensitive)");
* [PLEXOS_XML_FILE]       PLEXOS input database (xml) to be modified");
* [SUFFIXE]               (optional) Suffixe to be appended to the resulting modified PLEXOS input database. (Default=MOD)
